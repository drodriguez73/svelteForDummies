[Svelte Precios Crypto | Svelte & CoinGecko API](https://www.youtube.com/watch?v=uGhiM19d950) y arrancamos el tutorial para cosnumir datos desde la [API CoinGecko](https://www.coingecko.com/en/api/documentation)
