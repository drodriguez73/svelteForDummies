# Maqueta BOT

Aca tiramos todo el diseño estatico para el bot. Es donde hago las practicas de css al mismo tiempo que la pagina del Bot va tomando forma. El css de la maqueta utiliza el grid layout.

Idea Principal
La maquete tiene 3 secciones:
- Precios
- Nueva Orden
- Mis Ordenes

Cada diseño esta representado con un html + css y la home une todo el codigo para mostrarlo todo junto y ver como queda el final.
